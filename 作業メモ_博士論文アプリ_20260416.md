# 博士論文AIアシスタント 作業メモ
作成日：2026年4月16日

---

## 概要
老化防止医療の研究者（知人数名）向けに、博士論文作成を支援するAIアプリのプロトタイプを作成した。
ライフ・メモナビの既存バックエンド（life-memo-navi-backend on Render）とOpenAI APIを活用。

## 次にやること：Renderへのデプロイ

### GitHubリポジトリ
`https://github.com/MitsunoriFukuchi692/life-memo-navi.git`

### デプロイ手順（2通り）

#### 方法A：GitHubにpushして自動デプロイ（推奨）
1. VSCode か ターミナルで以下を実行：
   ```
   cd C:\Users\mfuku\._life-memo navi _0217
   git add backend/src/routes/thesis.ts backend/src/index.ts
   git commit -m "feat: 博士論文AIアシスタント用APIルート追加"
   git push origin main
   ```
2. RenderがGitHubと連携していれば自動でデプロイが始まる
3. Renderダッシュボードでデプロイ完了を確認（5〜10分）

#### 方法B：Renderダッシュボードから手動デプロイ
1. https://dashboard.render.com にログイン
2. `life-memo-navi-backend` サービスを開く
3. 右上の「Manual Deploy」→「Deploy latest commit」をクリック

### デプロイ後の確認
- `https://life-memo-navi-backend.onrender.com/api/thesis/chat` にPOSTリクエストが通れば成功
